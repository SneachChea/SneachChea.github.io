Place your font files here (e.g., JetBrainsMono-Regular.ttf).

Recommended files:
- JetBrainsMono-Regular.ttf
- JetBrainsMono-Regular.woff2 (preferred for modern browsers)
- JetBrainsMono-Regular.woff

After adding the files, run `bundle exec jekyll serve` and refresh your browser (clear cache if needed).

Make sure you have the right to distribute the font and check its license before publishing the site.
